Getting Started In order for other people to use (/)slash commands, you need to give @everyone the Use Slash Commands permission. You can prevent commands from being used in specific channels you don't want by disabling it in that channel. (Channel Settings > Permissions > @everyone > disable Use Slash Commands) To get a list of all of the commands, type: - /help If there is any questions/feedback/errors, join the support server and tell me!
